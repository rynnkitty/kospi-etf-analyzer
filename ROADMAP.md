# ROADMAP.md — 개발 로드맵

> 4주 5 Phase 개발 계획

---

## Phase 1: 프로젝트 초기화 + 데이터 파이프라인 (Week 1)

### 목표

프로젝트 기반 설정 + KRX 데이터 자동 수집 파이프라인 완성

### 마일스톤

- [M1] Next.js 프로젝트 초기화 + 전체 개발 환경 설정 완료
- [M2] TypeScript 타입 정의 + 상수 파일 작성 완료
- [M3] Python KRX 크롤링 스크립트 4종 작성 + 테스트 통과
- [M4] JSON 데이터 파일 생성 + 무결성 검증 통과

### 완료 조건

- `npm run build` 성공
- `python merge_data.py` 검증 통과
- `public/data/` 에 4개 JSON 파일 생성

---

## Phase 2: 프론트엔드 MVP (Week 2)

### 목표

핵심 4개 페이지(대시보드, 섹터, ETF, 종목) 구현 + JSON 데이터 연동

### 마일스톤

- [M5] 데이터 로딩 유틸리티 + 커스텀 훅 + Zustand 스토어 완성
- [M6] 공통 컴포넌트(Navbar, Footer, ValueBadge) + 루트 레이아웃 완성
- [M7] 대시보드 페이지 (섹터 카드 + 비교 차트 + 히트맵) 완성
- [M8] 섹터 상세 페이지 + ETF 상세 페이지 완성

### 완료 조건

- 대시보드에서 10개 섹터 카드 + 차트 표시
- 섹터 → ETF → 보유종목 탐색 경로 동작
- 보유종목별 PER/PBR 표시 + 색상 코딩

---

## Phase 3: UX 강화 + 검색/필터 (Week 3)

### 목표

전체 종목 탐색 + 필터/정렬/검색 + 반응형 최적화

### 마일스톤

- [M9] 전체 종목 탐색 페이지 + 필터 패널 완성
- [M10] 반응형 디자인 3단계(모바일/태블릿/데스크탑) 최적화 완료

### 완료 조건

- PER/PBR 범위 필터, 섹터 필터, 검색이 동작
- 모바일/태블릿/데스크탑에서 레이아웃 깨짐 없음

---

## Phase 4: 테스트 + CI/CD (Week 4)

### 목표

자동화된 테스트 + GitHub Actions CI/CD 파이프라인 완성

### 마일스톤

- [M11] 단위 테스트 작성 + 전체 통과
- [M12] GitHub Actions 워크플로우 2종 (데이터 수집 + 배포) 완성
- [M13] E2E 테스트 작성 + 전체 통과
- [M14] 최종 빌드 + 배포 테스트 성공

### 완료 조건

- `npm run lint` — 에러 0개
- `npm run test` — 전체 통과
- `npm run build` — 성공
- GitHub Actions 워크플로우 정상 동작
- GitHub Pages 배포 정상

---

## Phase 5: 확장 (선택, Week 4+)

### 가능한 확장

- 다크모드 (next-themes)
- PER/PBR 히스토리 차트 (시계열 데이터 축적)
- 포트폴리오 중복 분석 (/compare 페이지)
- 즐겨찾기/관심 종목 (localStorage)
- PWA 지원 (오프라인 접근)
- 데이터 내보내기 (CSV 다운로드)
